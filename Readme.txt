This system was developed by CodeIgniter, the purpose of this system development is to integrate data on high school students in the pesantren [Islamic boarding schools] that separate male students by female students. This system was developed by architecture of representational state transfer to exchange data

Requirements

1. PHP 5.1+
2. CodeIgniter 2.0.0+
3. cURL
4. CodeIgniter Curl library: http://getsparks.org/packages/curl/show
